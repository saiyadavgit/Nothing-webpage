Nothing Inspired Interactive Experience.

A cinematic and immersive frontend experience inspired by the design philosophy of nothing.tech⁠
Built to explore high-end motion design, real-time 3D interactions, and modern web animation techniques using a performance-focused frontend stack.

✨ Features
Real-time 3D model interaction with mouse tracking.
Smooth GSAP + ScrollTrigger animations.
Custom magnetic cursor system.
Glyph-inspired animated interface.
Scroll velocity based skew effects.
Frame-by-frame canvas animation sequences.
Responsive and modern UI experience.
Smooth transitions and cinematic motion design.

🛠️ Tech Stack
React.js
Three.js
GSAP
ScrollTrigger
JavaScript
HTML5 Canvas
Tailwind CSS / CSS3

🎯 Goal
The goal of this project was not just to create a website, but to build a digital product experience that feels interactive, reactive, and visually alive.
This project focuses heavily on:
Motion Design
Creative Frontend Development
Interactive UI Systems
Performance-focused Animations
